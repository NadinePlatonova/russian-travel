# Travel in Russia

_An adaptive single page application about Russian cities._

![header screenshot](./images/preview.png)

**Stack:**

- HTML;
- CSS.

**Realization:**

- flex and grid layouts;
- media queries for the site adaptability;
- nested BEM-methodology.
